### 시간이 겹칠까?

```python
import random

file = open('generate_random.txt', 'w')
N = random.randrange(2, 10000)
file.write(str(N) + '\n')

for i in range(N) :
    S = random.randrange(1, 1000000) #백만
    E = random.randrange(1, 1000000) #백만

    if S > E :
        S, E = E, S
    file.write(str(S) + ' ' + str(E) + '\n')
